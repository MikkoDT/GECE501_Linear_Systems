--Texlive Readme Offline Installation--

Requirements:

7-Zip or any other zip opener program (Zip Opener)

1. Copy and paste the link below to your browser to download.
   
2. https://ctan.ebinger.cc/tex-archive/systems/texlive/Images/texlive.iso

3. After downloading, open downloads folder ("C:\\Users\"Username"\Downloads")

4. Right click texlive.iso file (a context menu will open). hover to 7zip. Click extract to "\texlive"
 
5. After extracting open texlive folder and click "install-tl-windows.bat".
 
6. Right click and run as administrator. Click yes to all admin run prompts.
 
7. Install TexLive as it is.
